How to add this popup to any other button on your page

To attach the registration popup to any existing or new button (e.g., "Join Now", "Reserve Seat", "Get Access"), simply add the following JavaScript code or use the data-popup-trigger approach:

// Option 1: Add click listener to any button element
document.getElementById("yourButtonId").addEventListener("click", function(e) {
   e.preventDefault();
   openRegistrationModal("your-custom-source");
});

// Option 2: If button has a class, use querySelector:
document.querySelector(".any-register-btn").addEventListener("click", (e) => {
   e.preventDefault(); openRegistrationModal("class-trigger");
});

✅ The popup will not interfere with any other functionality and keeps its own styling, Google Sheets integration, and validation intact. Just call openRegistrationModal('sourceName') from your button's click handler.

Works for dynamically added buttons as well – attach event listener after creation.
